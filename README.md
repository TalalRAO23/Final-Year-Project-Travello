# Final-Year-Project-Travello
AI-powered tourism recommendation and booking platform developed as a Final Year Project. Helps users discover destinations, plan trips, and manage travel bookings through personalized recommendations.

## 📌 Project Overview

Travello aims to simplify travel planning by providing smart and personalized destination suggestions based on user preferences. The system allows users to explore tourist places, create travel plans, and manage their trip details in one place.

## ✨ Features

- AI-based destination recommendations
- Search and explore tourist locations
- Trip planning and itinerary management
- User authentication system
- Booking and travel management interface
- Responsive and user-friendly design
